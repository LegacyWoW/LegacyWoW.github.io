How to Play

LegacyWoW runs on the 3.3.5a client and progresses from Patch 1.0.0 → 3.3.5a with custom content along the way.

Go to Downloads

1) Set your realmlist

set realmlist 25.49.233.223

2) Create an account

Account creation is manual for now. Visit Create Account.

3) Select optional patches

Start patchmenu.exe to select any optional patches you wish to use, such as HD models, new mounts, or other available features.

4) Update and launch the game

Start WoWLauncher.exe to update the game as needed. Once finished, launch wow.exe to begin playing.
