FAQ

Common questions and quick fixes.

What client do I need?
You need a clean World of Warcraft 3.3.5a client.
Where do I set the realmlist?
Edit realmlist.wtf (commonly in Data\enUS or Data\enGB) and set:
set realmlist 25.39.161.60
I set the realmlist but it keeps reverting. Why?
Some client repacks include a launcher that overwrites realmlist. Try:
  • Make sure you are editing the correct language folder (enUS vs enGB).
  • Run the game using Wow.exe instead of a custom launcher.
  • Right-click realmlist.wtf → Properties → uncheck “Read-only” if needed (or check it to prevent changes).
I can’t log in (wrong password / account not found).
Make sure you created your account on the Create Account page. If account creation isn’t connected to the game server yet, the website will tell you. (We can wire it up to AzerothCore when your backend endpoint is ready.)
What does “progressive from 1.0.0 → 3.3.5a” mean?
The server begins with classic-era content and progresses forward through patches/eras up to 3.3.5a, with custom content added along the way.
Can I transfer my character?
If character transfers are offered, they’ll be announced in News.
